# VaultZen

VaultZen is an open-source bookmarking application built to help you save, organize, and capture links seamlessly while browsing. You can self-host it on your own infrastructure to retain full control over your data and deployment.

> VaultZen is under active development. New features and improvements will continue to evolve over time.

## Community & Support

- Report bugs, ask questions, or suggest new features via [GitHub Issues](https://github.com/zaymorielvandrell/vaultzen/issues)
- Visit the official site at [vaultzen.dev](https://vaultzen.dev)
